## 例如：A跳转到B，需传递selectedGoods对象

#### **A页面：**
```javascript
uni.navigateTo({
　url:'/pages/index/settle?selectedGoods='+encodeURIComponent(JSON.stringify(this.selectedGoods))
})
```

#### **B页面：**

```javascript
onLoad(data) {
　console.log(JSON.parse(decodeURIComponent(data.selectedGoods)));
}
```
